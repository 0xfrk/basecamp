## リポジトリの作成

GitHub へアクセスして新規リポジトリを作成しましょう。

https://github.com/

1. 画面右上の「+」アイコンをクリックし「New repository」を選択します。
2. Create a new repository ページに遷移したら「Repository name」欄に任意のリポジトリ名を入力します。
3. リポジトリはデフェオルトで Public（全体に公開される）リポジトリのため、後悔したくない場合は「Private」に変更しましょう。
4. Git 管理から一部除外する必要のあるファイルがあるため「Add .gitignore」で「Node」を選択しましょう。
5. 設定が終わったら「Create repository」でリポジトリを作成しましょう。
