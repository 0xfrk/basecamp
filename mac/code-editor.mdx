## エディタのインストール

以下の手順に従ってエディタをインストールします。<br />
※すでにインストール済みの方はこの手順は不要です。

### 1. インストーラーのダウンロード

Cursor の HP にアクセスし、トップページにある「Download for free」をクリックしてインストーラーをダウンロードします。

https://www.cursor.com/

### 2. インストール

1. ダウンロードしたインストーラーを実行します。
2. インストールが完了したら Cursor を起動します。
3. 初期設定の画面が表示されるため **Language** 欄に **日本語** と入力し **Continue** をクリック。<br />
   ※**Language** 以外の設定はデフォルトのままで問題なし
4. 次に **VS Code Extensions** の画面が表示されたら **Use Extensions** をクリック。<br />
   ※表示された場合
5. 次に **Data Preferences** の画面が表示されますが自分が記述したコードのデータの取り扱いに対するオプトインの設定が可能なため、サーバーに保存したくない場合は **Privacy Mode** を選択して **Continue** をクリック。
6. 設定が完了したらユーザー登録画面が表示されるため、**Sign Up** をクリックしてユーザー登録を行います。<br />
   ※この際にメールアドレス、Google アカウント、GitHub アカウントのいずれかで登録可能ですが、GitHub アカウントの方が色々と便利なので作成した GitHub アカウントで登録します。
7. ユーザー登録が完了するとブラウザに **Log in to Cursor desktop?** と表示されるため、**Yes, Log In** をクリックします。
8. 設定が問題なく完了すれば Cursor が起動します。

### 3. エディタの設定

以下の手順に従ってエディタの設定を行います。

#### 1. ディスプレイ言語の変更

1. キーボードの `Cmd + Shift + P` でコマンドパレットを開き、検索欄に **Lang** と入力。
2. 検索結果の **Configure Display Language** をクリック。
3. 最初は **English** になっているがしばらく待つと選択肢に **日本語** が表示されるため **日本語** をクリック。
4. クリックすると設定を反映するために再起動を求められるため **再起動** をクリック。

※選択肢に **日本語** が表示されない場合は以下の方法をお試しください。

1. キーボードの `Cmd + Shift + X` で機能拡張を開き、検索欄に **Japanese** と入力。
2. 検索結果の **Japanese Language Pack for Visual Studio Code** をクリック。
3. インストールをクリックしてインストールを行う。
4. インストールが完了すると設定を反映するために再起動を求められるため **再起動** をクリック。

#### 2. アクティビティバーの設定

1. キーボードの `Cmd + ,` で設定を開き、検索欄に **Vertical** と入力。
2. 検索結果に **Workbench > Activity Bar: Orientation** が表示されるため、**horizontal** から **vertical** に変更する。
